| SUPABASE_SERVICE_ROLE_KEY | Server-only, never exposed to client — used only in Server Action for manager email lookup |
| RESEND_API_KEY | Server-only environment variable |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-13 | Tampering | submitInquiry org_id | mitigate | org_id is a hidden form field — server action re-validates it against the listing_id (query listing by id and confirm listing.org_id matches submitted org_id before insert) |
| T-03-14 | Denial of Service | spam inquiry submissions | accept | Phase 3 scope; rate limiting via Vercel WAF or middleware is a post-v1 concern |
| T-03-15 | Information Disclosure | SUPABASE_SERVICE_ROLE_KEY | mitigate | Key only used in Server Action (server-side, never bundled to client); actions file has 'use server' directive |
| T-03-16 | Spoofing | fake manager email in inquiry notification | accept | Email is sent TO the manager resolved from the DB, not from visitor — visitor cannot control the To: address |
</threat_model>
