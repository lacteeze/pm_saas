Query (join for full context):
```sql
SELECT
  payments.created_at,
  payments.amount,
  payments.method,
  payments.status,
  payments.notes,
  people.first_name || ' ' || people.last_name AS tenant_name,
  properties.address AS property_address,
  units.address AS unit_address
FROM payments
JOIN leases ON payments.lease_id = leases.id
JOIN people ON leases.tenant_id = people.id
JOIN units ON leases.unit_id = units.id
JOIN properties ON units.property_id = properties.id
WHERE payments.org_id = [caller's org_id]
ORDER BY payments.created_at DESC
```

Use Supabase `.select()` with the nested join syntax or raw SQL via `.rpc()` / `.from('payments').select('...')`. Build the query programmatically with the supabase client.
