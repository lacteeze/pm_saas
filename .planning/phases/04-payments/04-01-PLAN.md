Write four migration files. All follow the established pattern: CREATE TABLE, ENABLE ROW LEVEL SECURITY, CREATE POLICY blocks.

**0016_create_payments.sql**
Create `payments` table:
- id uuid PRIMARY KEY DEFAULT gen_random_uuid()
- org_id uuid NOT NULL REFERENCES organizations(id) ON DELETE CASCADE
- lease_id uuid NOT NULL REFERENCES leases(id) ON DELETE RESTRICT
- amount numeric(10,2) NOT NULL CHECK (amount > 0)
- method text NOT NULL CHECK (method IN ('stripe','etransfer','cheque','cash','bank_transfer'))
- status text NOT NULL DEFAULT 'recorded' CHECK (status IN ('recorded','pending_clearance','cleared','failed'))
- stripe_payment_intent_id text NULL
- cleared_at timestamptz NULL
- disbursable_after date NULL
- recorded_by uuid NULL REFERENCES people(id)
- notes text NULL
- created_at timestamptz NOT NULL DEFAULT now()

RLS: ENABLE ROW LEVEL SECURITY. Two policies:
1. "Managers and admins manage payments" — FOR ALL USING (org_id = (SELECT org_id FROM people WHERE user_id = auth.uid() AND active = true LIMIT 1) AND EXISTS (SELECT 1 FROM people WHERE user_id = auth.uid() AND active = true AND (role @> ARRAY['manager'] OR role @> ARRAY['admin'])))
2. "Tenants read own payments" — FOR SELECT USING (lease_id IN (SELECT id FROM leases WHERE tenant_id = (SELECT id FROM people WHERE user_id = auth.uid() AND active = true LIMIT 1)))

**0017_create_expenses.sql**
Create `expenses` table:
- id uuid PRIMARY KEY DEFAULT gen_random_uuid()
- org_id uuid NOT NULL REFERENCES organizations(id) ON DELETE CASCADE
- property_id uuid NOT NULL REFERENCES properties(id) ON DELETE RESTRICT
- description text NOT NULL
- vendor_cost numeric(10,2) NOT NULL CHECK (vendor_cost >= 0)
- billed_amount numeric(10,2) NOT NULL CHECK (billed_amount >= 0)
- expense_date date NOT NULL
- created_by uuid NULL REFERENCES people(id)
- created_at timestamptz NOT NULL DEFAULT now()

RLS: ENABLE ROW LEVEL SECURITY.
1. "Managers and admins manage expenses" — FOR ALL USING (org_id = (SELECT org_id FROM people WHERE user_id = auth.uid() AND active = true LIMIT 1) AND EXISTS (SELECT 1 FROM people WHERE user_id = auth.uid() AND active = true AND (role @> ARRAY['manager'] OR role @> ARRAY['admin'])))
No tenant/owner SELECT policy — vendor_cost must never be readable by owner/tenant queries (per D-11).

**0018_create_stripe_events.sql**
Create `stripe_events` table:
- id uuid PRIMARY KEY DEFAULT gen_random_uuid()
- stripe_event_id text NOT NULL
- event_type text NOT NULL
- processed_at timestamptz NOT NULL DEFAULT now()
- payload jsonb NOT NULL DEFAULT '{}'
- CONSTRAINT stripe_events_stripe_event_id_key UNIQUE (stripe_event_id)

No RLS user policies needed (written only by service-role in webhook handler). Still ENABLE ROW LEVEL SECURITY with a single "Service role only" policy using auth.role() = 'service_role'.

**0019_create_owner_statements.sql**
Create `owner_statements` table:
- id uuid PRIMARY KEY DEFAULT gen_random_uuid()
- org_id uuid NOT NULL REFERENCES organizations(id) ON DELETE CASCADE
- property_id uuid NOT NULL REFERENCES properties(id) ON DELETE RESTRICT
- period_year int NOT NULL
- period_month int NOT NULL CHECK (period_month BETWEEN 1 AND 12)
- pdf_path text NOT NULL
- rent_collected numeric(10,2) NOT NULL DEFAULT 0
- total_expenses numeric(10,2) NOT NULL DEFAULT 0
- management_fee numeric(10,2) NOT NULL DEFAULT 0
- net_to_owner numeric(10,2) NOT NULL DEFAULT 0
- generated_at timestamptz NOT NULL DEFAULT now()
- generated_by uuid NULL REFERENCES people(id)
- CONSTRAINT owner_statements_property_period_key UNIQUE (property_id, period_year, period_month)

RLS: ENABLE ROW LEVEL SECURITY.
1. "Managers and admins manage statements" — FOR ALL USING same manager/admin pattern as above.
2. "Owners read their property statements" — FOR SELECT USING (property_id IN (SELECT id FROM properties WHERE owner_id = (SELECT id FROM people WHERE user_id = auth.uid() AND active = true LIMIT 1)))
